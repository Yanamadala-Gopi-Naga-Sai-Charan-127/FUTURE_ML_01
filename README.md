# Sales & Demand Forecasting using Machine Learning

## Project Overview
This project predicts weekly sales using historical Walmart sales data using Machine Learning.

## Objective
Develop a forecasting model to predict future weekly sales and help businesses make informed decisions.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Machine Learning Model
Random Forest Regressor

## Results
- Mean Absolute Error (MAE): 1392.68
- Mean Squared Error (MSE): 14861679.81
- R² Score: 97.15%

## Dataset
Walmart Sales Forecast Dataset
